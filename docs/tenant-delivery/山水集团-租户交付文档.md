## 附录：数据存储

| 项目 | 内容 |
|------|------|
| 系统库 | `data/_system.db` |
| 租户库 | `tenants/{tenantId}/data/tenant_{tenantId}.db` |
| 应用 Schema | `tenants/{tenantId}/apps/{appId}/` |

> 所有资源 ID 格式为 `{前缀}_{8位hex}`，如 `tenant_a1b2c3d4`、`app_5e6f7a8b`、`user_9c0d1e2f`。
> 前缀标识资源类型，便于按类型查找和更新。

### 租户目录结构

```
tenants/{tenantId}/
├── tenant.json                 # 租户元数据（时间戳格式）
├── apps/{appId}/               # 应用 Schema
│   ├── app.json                # 应用元信息（时间戳格式）
│   ├── pages/
│   ├── tables/
│   └── ...
└── data/                       # SQLite 数据库
    └── tenant_{tenantId}.db
```

> 所有 JSON 元数据中的时间字段统一使用 Unix 毫秒时间戳。

### 应用 Schema 文件清单

```
tenants/{tenantId}/apps/{appId}/
├── app.json                    # 应用元信息
├── pages/
│   ├── home.json               # 首页
│   ├── approval-list.json      # 审批列表
│   └── my-todos.json           # 我的待办
├── tables/
│   ├── approval-records.json   # 审批记录表
│   └── user-info.json          # 用户信息表
├── workflows/
│   └── procurement-approval.json  # 采购审批流程
├── cards/                      # 卡片（待创建）
├── forms/                      # 表单（待创建）
```

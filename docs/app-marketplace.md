## 审核流程

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│ 发布者提交    │────▶│ 自动检测      │────▶│ 人工审核      │
│ 模板审核      │     │              │     │              │
└──────────────┘     └──────────────┘     └──────────────┘
                            │                     │
                     ┌──────▼──────┐       ┌──────▼──────┐
                     │ 自动拒绝？   │       │ 审核结果     │
                     │             │       │             │
                     │ · 恶意代码   │       │ · 通过→上架  │
                     │ · 违规内容   │       │ · 拒绝→退回  │
                     │ · 依赖缺失   │       │ · 需修改     │
                     └─────────────┘       └─────────────┘
```

### 自动检测项

| 检测项 | 说明 |
|-------|------|
| Schema 完整性 | JSON Schema 格式校验，必要字段完整 |
| 组件兼容性 | 引用的组件在目标组件库中存在 |
| 依赖检查 | 引用的实体、数据源是否存在 |
| 内容安全 | 文本内容不含违规信息 |
| 安全扫描 | 自定义脚本不含危险代码（eval、fetch 等） |

### 人工审核要点

| 审核项 | 说明 |
|-------|------|
| 功能完整性 | 模板是否能正常运行，页面是否完整 |
| 体验质量 | UI 是否美观，交互是否合理 |
| 描述准确性 | 名称、描述、截图是否与实际一致 |
| 分类准确性 | 分类和标签是否恰当 |

### 审核状态流转

```
draft (草稿)
  │
  ▼ 提交审核
pending (待审核)
  │
  ├──▶ reviewing (审核中)
  │      │
  │      ├──▶ published (已上架)
  │      │
  │      ├──▶ rejected (已拒绝) ──▶ 修改后重新提交
  │      │
  │      └──▶ revision_needed (需修改) ──▶ 修改后重新提交
  │
  └──▶ 自动检测失败 ──▶ rejected
```

## 搜索与推荐

### 搜索能力

| 维度 | 说明 |
|------|------|
| 关键词搜索 | 模板名称、描述、标签全文搜索 |
| 分类筛选 | 按业务分类筛选 |
| 标签筛选 | 按标签多选筛选 |
| 排序 | 按安装量、评分、最新发布、最近更新排序 |
| 定价筛选 | 免费 / 付费 / 全部 |
| 组件库筛选 | 按兼容的组件库筛选 |

### 推荐策略

| 推荐位 | 策略 |
|-------|------|
| 首页精选 | 编辑推荐 + 高评分 |
| 热门模板 | 按近 30 天安装量排序 |
| 新上架 | 按发布时间倒序 |
| 相似推荐 | 基于当前模板的分类和标签推荐相似模板 |
| 个性化推荐 | 基于用户行业、已安装模板推荐 |

### 评论管理

- 评论需审核（防止恶意评论）
- 发布者可回复评论
- 用户可举报不当评论
- 安装过该模板的用户才能评论

## 模板版本管理

### 版本语义化

采用语义化版本号 `MAJOR.MINOR.PATCH`：

| 变更类型 | 版本号变化 | 示例 |
|---------|----------|------|
| 重大变更（不兼容） | MAJOR +1 | 1.0.0 → 2.0.0 |
| 新增功能 | MINOR +1 | 1.0.0 → 1.1.0 |
| Bug 修复 | PATCH +1 | 1.0.0 → 1.0.1 |

### 版本升级流程

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│ 发布者发布    │────▶│ 消费者收到    │────▶│ 消费者选择    │
│ 新版本        │     │ 升级通知      │     │ 升级时机      │
└──────────────┘     └──────────────┘     └──────┬───────┘
                                                  │
                                                  ▼
                                          ┌──────────────┐
                                          │ 兼容性检测    │
                                          │              │
                                          │ · 新增字段    │ → 自动合并
                                          │ · 删除字段    │ → 提示确认
                                          │ · 修改类型    │ → 冲突处理
                                          └──────┬───────┘
                                                  │
                                                  ▼
                                          ┌──────────────┐
                                          │ 应用升级      │
                                          │ 保留用户数据   │
                                          │ 更新模板配置   │
                                          └──────────────┘
```

## 与现有模块的关系

| 模块 | 关系 |
|------|------|
| **应用管理** | 模板本质是应用的导出 Schema，安装模板即创建应用 |
| **渲染引擎** | 模板包含页面 Schema，安装后可在设计器中编辑 |
| **权限引擎** | 模板发布/审核独立权限，市场浏览权限可控制 |
| **多租户** | SaaS 模式下模板市场全局共享，私有化部署支持私有市场 |
| **审计日志** | 模板发布、安装、下架操作记录审计日志 |
